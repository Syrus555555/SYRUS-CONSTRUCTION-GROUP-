# Syrus Construction Group Marketing Website UI Kit

## Overview
A 5-page click-through prototype of the Syrus Construction Group marketing website.
Pages: Home, Services, Work, About Us, Contact.

## File
- `index.html` — full interactive prototype (React/JSX, single file)

## Design Decisions
- Fixed top nav with frosted glass effect on scroll
- All-caps Bebas Neue display headings throughout
- Sharp zero-radius corners on all elements
- Section alternation: black ↔ white for rhythm
- Stone warm accent (#ccc4bb) used for labels and icons
- Ticker strip for service category animation
- Placeholder images marked for photography swap-in

## Components Included
- `Nav` — fixed top nav, active state, mobile menu (visual only)
- `Footer` — 4-column with links
- `HomePage` — hero, ticker, services preview, feature strip, work preview
- `ServicesPage` — full 6-service grid + CTA band
- `WorkPage` — filterable 3-col project grid
- `AboutPage` — leadership bios, stats, values
- `ContactPage` — contact info + form with success state
- `ImgPlaceholder` — grey rectangle placeholder for photography
- `Icon` — inline Heroicons (outline, 1.5px stroke)
